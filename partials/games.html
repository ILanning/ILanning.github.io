Outpost Update [Source]

    An in progress remake of a citybuilder first released in 1994. Most of the effort thus far has been put into developing a UI system that resembles the original game's use of Windows subwindows, without actually relying on the Windows API.

    An example of this window system is shown in one of the images to the right, displaying window ordering based on the last thing intereacted with, binding of window buttons to other objects, and multiple windows retrieving data from the same arbitrary data structure (In this case, a two dimensional string array).

Session Viewer [Source]

    This is an older project that I've been updating to more modern frameworks, from the abandoned XNA to the open-source Monogame. It allows you to explore the broad overview of a game session from the webcomic Homestuck. All non-UI data is brought in through XML, allowing users to create their own sessions with relative ease.

    The key programming hurdles here were developing elegant ways to manage changes to graphical objects over time, as well as finding good ways to handle input into a semi-complicated UI when no real UI features were provided. The program has a timeline feature which allows you to watch as things change in the session, and also allows you to skip around. This means that animations had to have a way to be stopped and skipped gracefully, which was a challege to figure out.

Tri-Layer Tower Defense

    Another old XNA project, which unfortunately I haven't had time to update to a GitHub-ready state. This was my first testbed for 3D graphics back in DigiPen's WaNIC program. The logic in this case is purely two dimensional, but it still makes use of a 3D play field. It's mostly a standard Tower Defense game, with the exeption that the enemies must path through 3 stacked layers in order to reach their goal. There are laser towers which attack enemies with a visible beam, and conjuration towers which allow you to place roadblocks. A path is calculated using A* whenever a tower is placed. If you try to place a tower in a location that would block all paths, the game prevents the placement.
